# Security Policy

## Supported Versions

| Version | Supported |
|---|---|
| 1.x (current) | ✓ |
| < 1.0 (beta) | ✗ |

## Reporting a Vulnerability

Jika kamu menemukan celah keamanan di proyek ini, mohon **jangan buat issue publik**.

Laporkan langsung melalui email ke:
**faris651234@gmail.com**

Sertakan informasi berikut dalam laporan:
- Deskripsi singkat tentang vulnerability yang ditemukan
- Langkah-langkah untuk mereproduksi masalah
- Dampak potensial yang bisa ditimbulkan
- Versi yang terpengaruh (jika diketahui)

Laporan akan direspons dalam **3–5 hari kerja**. Setelah diverifikasi, perbaikan akan dirilis secepat mungkin dan kamu akan diberitahu hasilnya.

Terima kasih sudah membantu menjaga keamanan proyek ini.
